Programming Language : JSP

# 2.개요
프로그래밍 관련 소식 및 정보 , 기술를 공유하는 커뮤니티 사이트    
# 3.필수기능 
로그인시 세션값으로 구분하여 로그인버튼이 로그아웃버튼으로 변함    
로그인시 게시물 작성이 가능하며 비로그인시 게시물 권한 수정 및 삭제 불가능    
게시글의 인덱스값을 이용하여 해당 게시물의 따라 상세페이지 출력    
게시글 작성시 특수기호 입력가능
